Common career paths

Path A — IT to security
From sysadmin to security engineer
1CompTIA Security+ (3–6 months)
2SOC analyst role (1–2 years)
3Specialise in SIEM/SOAR or cloud
4Security engineer / senior analyst
5CISSP + team lead / architect
Path B — non-tech career changer
From any background into GRC
1CompTIA Security+ or CISM foundations
2ISO 27001 Lead Implementer course
3Junior GRC analyst / compliance role
4CISM or CRISC certification
5GRC manager / DPO / vCISO
Path C — software dev to AppSec
From developer to application security
1OWASP Top 10 self-study + CTFs
2eJPT or CompTIA PenTest+
3Bug bounty hunting (HackerOne / Bugcrowd)
4Junior AppSec / pen tester role
5OSCP + senior pen tester / AppSec engineer
Path D — graduate entry
No experience to first role in 12 months
1Google Cybersecurity Certificate or TryHackMe
2CompTIA Security+ exam
3Home lab + GitHub portfolio
4Volunteer / intern at local SME
5Junior SOC analyst or helpdesk security
